# Predictive Analytics Project

This project demonstrates an end-to-end predictive analytics workflow using Python.
It includes data preprocessing, exploratory data analysis, supervised machine learning models,
and evaluation using standard performance metrics.

## Objectives
- Data cleaning and preprocessing
- Exploratory data analysis
- Supervised model building
- Model evaluation

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Project Workflow
1. Data preprocessing
2. Exploratory data analysis
3. Feature engineering
4. Model training
5. Model evaluation

## How to Run
Open `Int234 project.ipynb` and run the cells sequentially.
